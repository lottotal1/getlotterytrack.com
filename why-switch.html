Lottery Track didn't start in a boardroom. It started behind a counter, watching employees spend 20 minutes every night doing the same repetitive count — book by book, ticket by ticket, adding it all up on a calculator, sometimes having to start over because of a mistake.

We knew there had to be a better way. The barcode is right there on every book. The math is the same every night. So we built the tool that should have existed years ago.
